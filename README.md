# AutoClipGrabber
This tool allows one to insert a video or stream VoD into the code and auto generate clips from said video. 


Requirements: 

Install ffmpeg (required):
<b>Windows: https://ffmpeg.org/download.html </b>

All this tool does is detect loud parts of the video and pulls them. This isn't super accurate when looking for funny moments and stuff like that, but it does exelirate the editinfg process. On top of that, if you have a podcast or video format that is all talking. You could use this tool with the correct settinggs. In order to delete all the blank spaces and get the clips you need.
